About

This is an API I created for flags as PNG files. It includes both international flags and Israeli flags.

Current international flags available:

  • Countries
  • Pride
  • US States
  • Nautical
  • International Organizations

Current Israeli flags available:

  • Governmental
  • Coat of Arms
  • Historical
  • Municipalities
  • Organizations
  • Security Forces (IDF, Police, Fire and Rescue, etc)
  • Universities

How to use

The response will be a JSON that includes the URL of requested images.

  • In order to get all the images of a category, use the following query:
    /get_category/?all={category}.
    The content will be a list of all the URLs of PNG images in that category.
  • In order to get a certain image of a category, use the following query:
    /get_image/?category={category}&image={image}.
    The content will be the URL of the requested PNG image.
